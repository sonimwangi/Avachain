# Avachain

Avachain is a play-to-learn Web3 game built on Avalanche that helps users understand blockchain concepts through interactive gameplay and on-chain rewards.

## What problem are we solving?
Web3 onboarding is still complex and inaccessible for many beginners. Most learning resources are text-heavy and not engaging.

## Our solution
Avachain turns blockchain education into a game. Players learn by playing, completing levels, and earning on-chain rewards on Avalanche.

## Why Avalanche?
- Fast finality
- Low transaction fees
- Developer-friendly
- Ideal for gaming and education use cases

## Current Status
MVP in development for the Avalanche competition.
